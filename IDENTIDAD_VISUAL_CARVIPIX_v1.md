## MOVIMIENTO Y ANIMACIÓN

### Regla de Oro:
```
CADA MOVIMIENTO DEBE RESPONDER:
"¿Qué está pasando en la realidad que se vea así en pantalla?"

Si no hay respuesta: no se mueve.
```

### Velocidades de animación:

```
RÁPIDO (Respuesta a usuario):
Duración: 200ms - 400ms
Easing: cubic-bezier(0.34, 1.56, 0.64, 1) [bounce]
Uso: Click, hover, cambios inmediatos
Sensación: Responsivo, vivo

MEDIO (Cambio de datos):
Duración: 600ms - 1000ms
Easing: cubic-bezier(0.25, 0.46, 0.45, 0.94) [smooth]
Uso: Número cambia, línea se fortalece, panel actualiza
Sensación: Suave, controlado, profesional

LENTO (Proceso completado):
Duración: 1200ms - 2000ms
Easing: cubic-bezier(0.17, 0.67, 0.83, 0.67) [ease-out]
Uso: Análisis completado, decisión tomada, pulsación
Sensación: Reflexivo, importante, significa algo
```

### Tipos de movimiento:

```
ROTACIÓN:
- ¿Qué significa? Proceso activo, cálculo ocurriendo
- Velocidad: Indica intensidad del proceso
- Nunca detiene (comunica: sigue funcionando)
- Ejemplo: Rueda de mercado gira según volatilidad

CAMBIO DE BRILLO (Glow):
- ¿Qué significa? Actividad ahora mismo
- Intensidad: Más intenso = más importante
- Pulsaciones: Comunican latencia (algo vivo)
- Ejemplo: Centro brilla cuando análisis se completa

DESPLAZAMIENTO:
- ¿Qué significa? Dato moviéndose en el tiempo
- Dirección: Siempre tiene significado (arriba/abajo/izq/der)
- Suavidad: Comunica fluidez o fricción
- Ejemplo: Nueva vela ingresa al gráfico desde derecha

ESCALADO:
- ¿Qué significa? Cambio de importancia o focus
- Proporción: Suave (1.0 → 1.1), nunca radical
- Timing: Coordina con contexto
- Ejemplo: Número sube 0.5% en importancia cuando destaca

FADE (Opacidad):
- ¿Qué significa? Elemento entra/sale de relevancia
- Velocidad: Acorde a importancia del cambio
- Nunca es 0 (nunca desaparece instantáneamente)
- Ejemplo: Evento antiguo desvanece en timeline
```

### NO es decorativa. Es FUNCIONAL.

```
Concepto: CARVIPIX es como un centro de operaciones.
La pantalla emite luz (como monitor real).
La luz comunica estado.

Iluminación tiene tres niveles:
```

### Nivel 1: BASE (Fondo)
```
Color: Negro puro (#000000) o azul muy oscuro (#0B111A)
Función: Máximo contraste, cero distracciones
Sensación: "Esto es serio"
Efecto: Hace que todo lo demás brille más
```

### Nivel 2: ESTRUCTURAL (Paneles)
```
Color: Dorado sutil alrededor de elementos
Intensidad: 20-30% de opacidad
Función: Define límites, agrupa información
Sensación: "Esto está contralado"
Técnica: Border dorado + shadow interno sutil
```

### Nivel 3: ACTIVIDAD (Elementos)
```
Color: Dorado brillante (procesando), Blanco (activo)
Intensidad: 40-60% de opacidad
Función: Muestra qué está pasando AHORA
Sensación: "Algo está sucediendo"
Técnica: Glow + pulsaciones suaves
```

### Reglas de iluminación:
```
1. El brillo SIEMPRE significa actividad
2. Mayor brillo = mayor importancia
3. Brillo pulsante = proceso en progreso
4. Brillo constante = estado activo
5. Sin brillo = pasivo (pero existe, no oculto)

Nunca:
- Decorativo
- Parpadeo rápido
- Múltiples luces compitiendo
- Colores vibrantes
```
