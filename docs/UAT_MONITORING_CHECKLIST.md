# CARVIPIX UAT Monitoring Checklist

## Runtime
- Health checks periodicos activos.
- Alertas sobre errores 5xx activas.
- Alertas sobre caida de DB/bridge activas.

## Security
- Monitoreo de intentos no autorizados admin.
- Verificacion de logs sanitizados.

## Product Signals
- Verificar actualizacion de dashboard y modulos criticos.
- Verificar coherencia de origen de datos (REAL/SANDBOX/DEMO/MOCK).

## Daily UAT Review
- Resumen de incidentes por severidad.
- Estado de pruebas bloqueadas.
- Riesgos abiertos para siguiente ciclo.
