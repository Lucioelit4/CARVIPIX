# CARVIPIX Checklists Empresariales

Documentos oficiales para fase de preproduccion empresarial:

- `PREPRODUCTION_CHECKLIST.md`: validacion integral antes del primer despliegue real.
- `GO_LIVE_CHECKLIST.md`: secuencia controlada para publicar una nueva version.
- `RECOVERY_CHECKLIST.md`: respuesta y recuperacion ante incidentes operativos.
- `SCALABILITY_CHECKLIST.md`: preparacion de crecimiento sin romper arquitectura.

Regla transversal:
- Procesamiento pesado en estacion local.
- Servidores de produccion orientados a trafico de clientes y continuidad de servicio.
